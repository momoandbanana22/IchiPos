# IchiPos

IchiPos は、Windows 11 上で動作するコマンドラインアプリケーションです。

Misskey をメイン投稿先、X をサブ投稿先として、投稿テキストを同時に投稿準備します。

## 機能

- 投稿テキストを取得する（文字列または `.txt` ファイル）
- Misskey に投稿する
- X の投稿画面をブラウザで開く

## 動作環境

- OS：Windows 11
- 言語：C#
- フレームワーク：.NET 10
- アプリ形式：コンソールアプリケーション

## 実行形式

```text
IchiPos.exe <content> [--image-path <folder>]
```

## 設定ファイル

設定ファイルは YAML 形式で、実行ファイルと同じディレクトリの `./config` に配置します。

設定ファイルが存在しない場合はエラーとなります。
